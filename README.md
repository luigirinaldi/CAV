-------------------------------------------------------------------------------
**                                SMOKE TEST                                 **
-------------------------------------------------------------------------------

Run the following to load the Docker image:

  docker load < parabit-artifact.tar

After that, start an interactive session with:

  docker run -it parabit-artifact

This places you in a bash shell at `/artifact/`.

Run the smoke test with:

  ./scripts/run_smoke.sh                           [~ runtime: 15 minutes]

The smoke test runs parabit and pbv on the Alive and Hydra benchmarks with a
1-second timeout and 1 GB memory limit, then verifies a selection of proofs
with Isabelle.

If everything runs successfully you will see progress output from each stage,
ending with the Jupyter notebook executing without errors. All results are
written to the `smoke_test/` directory:

  smoke_test/
    parabit/Alive/results.csv      -- parabit results on Alive
    parabit/Hydra/results.csv      -- parabit results on Hydra
    parabit_verif/Alive/results.csv -- Isabelle-verified proofs for Alive
    pbv/Alive/results.csv          -- pbv results on Alive
    pbv/Hydra/results.csv          -- pbv results on Hydra
    plots/Figure7.pdf              -- survival plots (Alive + Hydra only)
    plots/Figure7_allproblems.pdf  -- union variant of Figure 7
    tables/table2/single.tex       -- Table 2 (single-width, Alive + Hydra)
    tables/table2/multiple.tex     -- Table 2 (multi-width, Alive + Hydra)

- To check that Table 2 generation worked, inspect:

      cat smoke_test/tables/table2/single.tex

    The table should contain rows for Alive and Hydra with non-zero solved
    counts in the pbv_full, parabit, and vbs columns.

- To check that Figure 7 generation worked, inspect:

      ls smoke_test/plots/

    The directory should contain Figure7.pdf and Figure7_allproblems.pdf.
    Each PDF should contain two survival plots (one per benchmark).

- To check that Isabelle proof verification worked, run:

      grep -c "verified,True" smoke_test/parabit_verif/Alive/results.csv

    The output should be a non-zero count of verified proofs.

-------------------------------------------------------------------------------
**                               FULL REVIEW                                 **
-------------------------------------------------------------------------------

Assuming the smoke test passed, start a fresh interactive session:

  docker run -it parabit-artifact

All results are written to `results/` and figures/tables to
`results/plots/` and `results/tables/` respectively.

We provide two versions: a short version with tighter resource limits that
completes in a few hours, and the full version matching the paper results.

**Short version** (5-second timeout, 2 GB memory per process):

  ./scripts/run_small.sh                           [~ runtime: FILL IN]

**Full version** (60-second timeout, 8 GB memory per process):

  ./scripts/run_full.sh                            [~ runtime: FILL IN]

Both scripts run parabit and pbv on all four benchmark suites (ROVER, Alive,
Hydra, Cadence), verify proofs with Isabelle, and generate all tables and
figures. Concrete timing values will differ from the paper on different
hardware, but the overall trends (relative solve rates, ranking of tools)
should remain consistent.

(1) To obtain the results in Table 2, inspect:

      results/tables/table2/single.tex
      results/tables/table2/multiple.tex

    These contain the per-benchmark breakdown of solved problems for parabit,
    pbv (full and restricted), and the virtual best solver.

(2) To obtain the results in Table 3, inspect:

      results/tables/table3.tex

    This contains per-benchmark averages for e-graph nodes, classes,
    iterations, solve time, proof length, and proof verification time.

(3) To obtain Figure 7, inspect:

      results/plots/Figure7.pdf
      results/plots/Figure7_allproblems.pdf

    These contain the cactus/survival plots for all four benchmarks, showing
    cumulative solve counts as a function of time for parabit, pbv, and VBS.
